A technical perspective

How we accomplished this

Language: Python (3)

Python is a widely used and adapted language, all while being a high-level language and simple. We chose python because it has libraries that were special to us,
Specifically, Django (HTML wrapper and server), PyCrypto (data encryption), PyODBC (Python database and connection).
Python also has support for GUIs and libraries that allow us to make it sleek and modern looking.
Deployment in Python is very simple, as our Django website runs by a simple command.

Backend: Microsoft Access Database

Microsoft Access is easy and reliable. It also supports communication with Python, so we can do database management through Python in SQL.
Access itself can be a front-end to a MySQL database and it then supports all of MySQLs huge usability.

Frontend: Python GUI
